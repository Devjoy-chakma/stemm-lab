# STEMMLab

A real-world science game for Years 3–9. Seven hands-on STEMM challenges captured through a mobile phone's sensors, with team-based leaderboards.

Built with React Native + Expo + TypeScript + Firebase for the Mobile Applications Development unit.

## Prerequisites

- Node.js 20 or higher
- npm 10 or higher
- Expo Go installed on your iOS or Android phone (free, from the App Store / Play Store)
- Laptop and phone on the same Wi-Fi network

## Getting started

1. Clone the repo
```bash
   git clone git@github.com:Devjoy-chakma/stemm-lab.git
   cd stemm-lab
```

2. Install dependencies
```bash
   npm install
```

3. Start the development server
```bash
   npx expo start
```

4. Scan the QR code in the terminal with your phone's Camera app. Tap the notification to open in Expo Go.

## Project structure

- `App.tsx` — root component
- `index.ts` — app entry point
- `app.json` — Expo configuration
- `tsconfig.json` — TypeScript compiler config
- `assets/` — app icons and splash images

## Scripts

- `npx expo start` — start the dev server
- `npm run ios` — start + open iOS simulator (requires Xcode)
- `npm run android` — start + open Android emulator
- `npm run web` — start + open in browser
